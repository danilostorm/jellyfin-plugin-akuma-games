Akuma Games v0.2.1

O plugin cria uma biblioteca nativa Games na página inicial do Jellyfin, disponível também para usuários comuns que tenham permissão de acesso à biblioteca.

Sem reprodução como vídeo

Nenhum arquivo .strm é criado. As capas e pastas funcionam como catálogo nativo, evitando FFmpeg, HLS e o erro fatal do player.

Fire TV e Android TV

A biblioteca e as capas aparecem no cliente nativo. A execução interativa do game HTML5 será feita pelo cliente Android TV compatível com a API Akuma Games.
